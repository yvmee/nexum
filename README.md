# nexum

An educational browser game prototype built with React 19.2, TypeScript and Tailwind.
This is a first prototype for a single player game based on Play x Discuss.

## Features

- Play through a scenario and make decisions.
- Reflect on your decisions and share your thoughts.
- Get feedback and guidance.
- See other player's reflections asynchronously and discuss with future and past players.

## Tech Stack

- **React 19.2** - Modern UI library
- **TypeScript 5.9** - Type-safe development
- **Vite 7.3** - Fast build tool and dev server
- **Tailwind CSS 4.1** - Utility-first CSS framework

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
